## 🤙 Git Commit Convention

|*Type*|*Subject*|
|:---|:---|
|**[Feat]**|새로운 기능 추가|
|**[Fix]**|버그 수정|
|**[Docs]**|문서 관련 추가, 수정, 삭제|
|**[Style]**|코드의 변화와 관련없는 포맷의 변경|
|**[Refactor]**|코드 리팩토링|
|**[Comment]**|필요한 주석 추가 및 변경| 
|**[Rename]**|파일 혹은 폴더명을 수정하거나 옮기는 작업| 
|**[Remove]**|파일을 삭제하는 작업 수행|
